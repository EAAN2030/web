Etiqueta Definicion
"b" Indica que el texto debe ser representado con una variable bold, o negrita,de la tipografía que se esté usando
"ol" El elemento ol representa una lista (o secuencia) de elementos; es decir, una lista en la que los elementos están ordenados intencionalmente, de modo que cambiar el orden cambiaría el significado de la lista.
"ul" El elemento ul define una lista desordenada. El elemento contiene uno o más elementos li que especifican los ítems de la lista. Por el contrario, el elemento ol define una lista ordenada. El elemento contiene uno o más elementos li que especifican los ítems de la lista como una secuencia ordenada.
"li" Definición. El elemento li del ingles item list o elemento de lista declara cada uno de los elementos de una lista. Sus etiquetas son: "li" y "/li" (la de cierre es opcional).
"table" El Elemento de Tabla HTML ( "table" ) representa datos en dos o mas dimensiones. Nota: Antes de la creación de CSS, los elementos HTML "table" eran frecuentemente utilizados para la disposición (posicionamiento) de una página. Este uso ha sido desalentado desde HTML 4, y el elemento no debe ser usado para dichos fines.
"tr" define una fila en una tabla HTML
"td" Define una columna en una tabla HTML
"a" Indica que se va a utilizar un hipervinculo
"p" Indica que es un parrafo de texto
"br" Indica un salto de Linea
